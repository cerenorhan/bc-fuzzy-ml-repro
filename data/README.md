# Data folder

This folder intentionally does not include the dataset.

## How to obtain the data
The dataset can be requested from the **corresponding author** of the associated manuscript.

## Expected file
Place the Excel file here with the exact name:
`AI_B_CANCER_STUDY_TNZ_Niyazi_CLEANED_STAGE2.xlsx`

The pipeline expects a sheet named `ModelReady`.
See `src/config.py` for expected columns.
